# Assessor Financeiro Pro 💰

Sistema web completo para gestão financeira pessoal, desenvolvido com tecnologias frontend modernas e zero dependências de backend.

## ✨ Funcionalidades

- 📊 **Dashboard Inteligente** com saldo, receitas, despesas e taxa de economia
- 📁 **Importação de Planilhas** (Excel/CSV) com drag-and-drop
- 💰 **Cálculo de Limites** diários, semanais e mensais de gastos
- 📈 **Gráficos Interativos** (Chart.js) - pizza, linha e barra
- 🎯 **Gestão de Metas** de curto, médio e longo prazo
- 💾 **Persistência Local** - dados salvos no navegador
- 📱 **Design Responsivo** - funciona em mobile e desktop

## 🚀 Como Usar

### Opção 1: Abrir Diretamente
1. Baixe todos os arquivos
2. Abra `index.html` no navegador (Chrome, Firefox, Edge, Safari)
3. Pronto! Funciona 100% offline
